# "Where Does the Model Look?"- Attention Across Modalities in Vision Language Understanding

## Usage:
- Create a virtual environment and run ```pip install -r requirements.txt```
- Make sure to have the subset of images ready in the images folder (a subset of images from the coco/train2017 folder)
- Run ```python script.py``` and you should expect to see the captions in the `captions.jsonl` file
